# EduERP - College Management System

A full-stack MERN application for managing students, attendance, fees, and leave requests.

Features:
✅ JWT Authentication
✅ Student Dashboard
✅ Staff Dashboard
✅ Attendance Management
✅ Fees Management
✅ Leave / OD Approval Workflow
✅ Department-based Access Control

Tech Stack:
- React.js
- Node.js
- Express.js
- MongoDB
